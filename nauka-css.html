Jak prawidłowo tworzyć szablon CSS?

Podobnie jak w przypadku HTML-a to sztuka, którą opanowało niewielu. Podstawa to systematyczność i pełna koncentracja plus kawa hahah nie ma mowy o szukaniu wymówek!

Czego nauczyłem się z lekcji?

Podstawowa składnia CSS:

selektor { właściwość: wartość; } np.: body{ background: lightblue; } h1{ color: red; } p{ color: black; } itp..
Arkusze stylów można umieścić w html w sekcji head>/head> używając polecenia style> .
Bądź umieścić jako zewnętrzny arkusz, także w sekcji head>/head>ze ścieżką, link href=”style.css” rel=”stylesheet” type=”text/css”>.
Nauczyłem się też używać narzędzi DevTools
